# Passive Components Study using LTspice

## Overview

This project focuses on the study and simulation of fundamental passive electronic components using LTspice.

The objective is to understand the electrical behavior of basic components before moving toward more complex analog electronic designs such as signal conditioning circuits, power supplies and embedded hardware systems.

This project is part of my Electronics Engineering Portfolio.

---

## Objectives

The main objectives of this study are:

- Understand the electrical behavior of passive components
- Validate theoretical calculations through simulation
- Analyze voltage, current and power relationships
- Learn simulation workflow using LTspice
- Develop a strong foundation in analog electronics

---

## Components Studied

### Resistors

Study of:
- Ohm's law
- Voltage divider circuits
- Power dissipation
- Series and parallel configurations

### Capacitors

Study of:
- Charging and discharging behavior
- RC circuits
- Time constants
- Filtering applications

### Inductors

Study of:
- Transient response
- RL circuits
- Energy storage
- Magnetic behavior
